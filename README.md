Blog built with [Eleventy](https://github.com/11ty/eleventy) static site generator.
  

### 2. Navigate to the directory
 

```
npm install
```

```
npx eleventy
```

Or build and host locally for local development
```
npx eleventy --serve
```

Or build automatically when a template changes:
```
npx eleventy --watch
```

Or in debug mode:
```
DEBUG=* npx eleventy
```
